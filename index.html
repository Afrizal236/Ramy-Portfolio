Tentang Project

RamySmart adalah aplikasi pembelajaran online yang dirancang khusus untuk membantu siswa dalam persiapan Seleksi Nasional Berdasarkan Tes (SNBT). Aplikasi ini dilengkapi dengan fitur-fitur modern seperti authentication system, payment gateway, progress tracking, dan interactive learning experience yang komprehensif.

🔐

Secure Authentication

Sistem autentikasi yang aman dengan Firebase Authentication, mendukung registrasi dan login yang mudah

💳

Payment Integration

Terintegrasi dengan payment gateway untuk pembelian course dengan berbagai metode pembayaran

📊

Progress Tracking

Fitur pelacakan progress belajar dengan sistem grading otomatis dan analytics yang detail

📱

Mobile First Design

Desain responsif yang dioptimalkan untuk pengalaman mobile yang smooth dan user-friendly

🎯

Interactive Learning

Sistem pembelajaran interaktif dengan video, quiz, assignment, dan kuesioner SNBT

🛒

E-Commerce Features

Fitur e-commerce lengkap dengan cart, wishlist, dan sharing untuk pengalaman belanja yang optimal

Tentang Project

Kalkulator API RA adalah aplikasi kalkulator pintar yang dibangun dengan Flutter dan mengintegrasikan API MathJS untuk perhitungan matematika yang kompleks. Aplikasi ini memiliki dua mode: mode biasa untuk operasi dasar dan mode scientific untuk fungsi trigonometri, logaritma, dan operasi matematika lanjutan. Dilengkapi dengan sistem fallback yang otomatis beralih ke perhitungan lokal ketika koneksi internet tidak tersedia.

🧮

Dual Mode Calculator

Mode biasa untuk operasi dasar dan mode scientific untuk fungsi matematika lanjutan

🌐

API MathJS Integration

Menggunakan API MathJS untuk perhitungan yang akurat dan mendukung berbagai fungsi matematika

📱

Offline Fallback

Sistem fallback otomatis menggunakan dart:math ketika koneksi internet tidak tersedia

📐

Trigonometric Functions

Fungsi sin, cos, tan dengan mode derajat dan radian yang dapat diubah

🔢

Advanced Math Functions

Logaritma, eksponen, akar kuadrat, pangkat, dan konstanta matematika (π, e)

⚡

Real-time Calculation

Perhitungan cepat dengan indikator loading dan status koneksi jaringan
